## ğŸš€ Instalasi & Setup

### Prasyarat
- PHP >= 8.2
- Composer >= 2.x
- Node.js >= 18.x
- NPM >= 9.x

### Langkah Instalasi

```bash
# 1. Clone repository
git clone https://github.com/fg120/tugas-akhir-pweb.git
cd tugas-akhir-pweb

# 2. Install dependencies
composer install
npm install

# 3. Setup environment
cp .env.example .env
php artisan key:generate

# 4. Setup database
touch database/database.sqlite
php artisan migrate

# 5. Seed data (opsional, untuk testing)
php artisan db:seed

# 6. Build assets
npm run build

# 7. Jalankan server
php artisan serve
```

### ğŸ”§ Quick Setup (Satu Perintah)

```bash
composer setup
```

### ğŸ”„ Development Mode

```bash
composer dev
```
Perintah ini akan menjalankan secara bersamaan:
- `php artisan serve` - Server Laravel
- `npm run dev` - Vite Hot Reload
